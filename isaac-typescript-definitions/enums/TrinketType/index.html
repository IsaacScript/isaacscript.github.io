TrinketType

For EntityType.PICKUP (5), PickupVariant.TRINKET (350).

This is the sub-type of a trinket.

This enum is not contiguous. In other words, the enum ranges from TrinketType.NULL (0) to TrinketType.SIGIL_OF_BAPHOMET (189), but there is no corresponding TrinketType with a value of 47.

Enumeration Members​

NULL​

• NULL = 0

Spawning trinkets with this sub-type will produce a random trinket.
