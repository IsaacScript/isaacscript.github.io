EntityFlag

This is represented as an object instead of an enum due to limitations with TypeScript enums. (We want this type to be a child of the BitFlag type.)

• NO_STATUS_EFFECTS = number

Prevents status effects from applying to the entity (e.g. freeze, poison, slow, charm, confusion, fear, burn, etc.).

RENDER_FLOOR​

• RENDER_FLOOR = number

Meant for entities that have a sprite loaded that represent a floor texture. Entities with this flag will be removed after their first render. (Removing the entity is desirable because we would not want it to override other things that render to the floor, like bomb explosions.) You can use EntityFlag.NO_REMOVE_ON_TEX_RENDER to disable this behavior.
